🏛️ Dispora Rebranding Final
Repositori ini merupakan bagian dari proyek rebranding website DISPORA Kota Semarang (Dinas Kepemudaan dan Olahraga), yang bertujuan untuk memperbarui tampilan dan meningkatkan pengalaman pengguna.

🚀 Fitur Utama
✅ Halaman Beranda responsif dan modern

✅ Profil Pegawai dinamis (dengan struktur organisasi)

✅ Sistem berita dan artikel terkini

✅ Tampilan peta interaktif menggunakan OpenStreetMap

✅ Dokumen publik yang dapat diakses (PDF viewer)

✅ Mode admin terpisah untuk pengelolaan konten

⚙️ Teknologi yang Digunakan
Frontend: React + Vite

Styling: Tailwind CSS

Animasi: Framer Motion

Peta: Leaflet (OpenStreetMap)

Library Tambahan: date-fns, react-router-dom, dll.
